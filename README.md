# RandomWalker

## これはなに
ランダムなお散歩経路を提案するアプリ

## Require
Google MapsのAPIキーを所定の場所に入れておくこと

## 機能
- ランダムな散歩コースを提案
  - 自分じゃ選ばないような道を進もう
- 所要時間をなんとなく指定可能
  - 1時間ぐらい散歩したいな、に対応
- お散歩する方角を指定可能
  - 8方位+選ばない+ランダムに対応
- カオスなルートも生成可能
  - 散歩上級者向け
- Google Mapsで開く機能
  - 歩きスマホはやめましょう
- URLエクスポート機能
  - 気に入った経路は保存しておこう
